## Description
A Machine Learning project that predicts whether a telecom customer is likely to churn (leave the service) based on customer demographics, account information, service subscriptions, and billing details. The model helps telecom companies identify at-risk customers and implement retention strategies to reduce customer loss.

* Customer churn prediction using Machine Learning
